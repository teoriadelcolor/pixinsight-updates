Bienvenido a Astro Processor Script for PixInsight

Este es el repositorio oficial de Astro Processor Script for PixInsight, una colección de herramientas y scripts para mejorar el procesamiento de imágenes astronómicas en PixInsight. Aquí encontrarás recursos útiles y actualizaciones para optimizar tu flujo de trabajo en el procesamiento de astrofotografías.
